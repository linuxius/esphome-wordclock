# Home Assistant Examples

This folder contains publish-safe example YAML files for Home Assistant.

## Files

- `ha_wordclock_auto_brightness.example.yaml` -> `automations.yaml`
- `ha_wordclock_preset_scripts.example.yaml` -> `scripts.yaml`
- `ha_wordclock_template_entities.example.yaml` -> package/template config
- `ha_wordclock_dashboard.example.yaml` -> Lovelace raw dashboard config

## Notes

- Entity IDs are generic placeholders (for example `light.word_clock` and
  `sensor.solar_radiation_wm2`).
- Update entity IDs to match your own Home Assistant setup before use.
